# simple-react-app #

A simple application for demo purpose using React.js

## Setup ##

Requirements:
- node >= v11.12.0
- npm >= 6.9.0
- create-react-app >= 2.1.8

Install application : `npm install`

## Useful command ##

Run dev environement : `npm start`

## Cheat sheet ##

create a new app : `npm install -g create-react-app`
